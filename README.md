## Features

- Secure Firebase Authentication
- Customer & Vehicle Profile Intake
- AI Diagnostic Engine
- OBD Fault Code Lookup
- Smart Invoice Builder
- Revenue Dashboard
- Workshop Reports History
- PDF Export
- Responsive Premium UI
- GitHub Pages Deployment
